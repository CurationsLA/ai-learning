Implementation Patterns

Consent-First:

// Don't do this
autoSubmitToAI(userData);

// Do this

  submitToAI(userData);
} else {
  processWithoutAI(userData);

// Mark AI-generated content

Graceful Degradation:

// AI fails? System still works

  result = await aiProcess(input);
